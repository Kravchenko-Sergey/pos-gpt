ПОЛНАЯ НАСТРОЙКА ТЕРМИНАЛА КАТЕГОРИИ IKR
(Актуально для SP30, S300, Q25, Ingenico и др., подключение по USB к ПК или кассе)

------------------------------------------------------------
1. ПОДГОТОВКА (ДО НАСТРОЙКИ ТЕРМИНАЛА)
------------------------------------------------------------
- Выполните настройку регформы в мобильном приложении (МП).
------------------------------------------------------------
Группа:
[*] Вариант связи
Шаблон:
U3 Ethernet [ИКР]

Группа:
Настройка параметров PCI DSS ИКР (без вариантов связи)
Шаблон:
ИКР. Без вар.связи. Краткий 1
------------------------------------------------------------
- Сохраните регформу.

------------------------------------------------------------
2. НАСТРОЙКА ТЕРМИНАЛА
------------------------------------------------------------
1. На терминале нажмите «Настроить».
2. Выберите вариант связи: «Новый» -> «Через ККМ» (т.к. подключение USB/COM).
   *Важно: Ethernet (RJ45) — только для работы с IP-адресами.*
3. Параметры по умолчанию (менять, только если требует техпаспорт, например X5 Group).
4. «Управление USB» -> пропустить (не важно).
5. Введите НОМЕР ТЕРМИНАЛА (строго из заявки).
6. Выберите канал связи пинпада: 
   - Вариант 2: «Пинпад через кассу» (верный для неполного цикла, чек не печатает, связь через кассу).
7. Нажмите зеленую кнопку. Терминал перезагрузится. ОТЛОЖИТЕ ЕГО В СТОРОНУ.

------------------------------------------------------------
3. НАСТРОЙКА КОМПЬЮТЕРА (ПАПКА sc552)
------------------------------------------------------------
1. На диске C: создайте папку (если нет): sc552 (строго маленькими буквами, название от банка).
2. Определите версию ПО терминала:
   - Выключите/включите питание терминала.
   - При загрузке увидите версию (пример: 33.18.03).
3. На флешке инженера найдите нужную версию (например, UPOS 33.18.03).
4. Скопируйте в C:\sc552 ТОЛЬКО ОДНУ из директорий:
- Для Linux:
   - LINUX 
   - LINUX_ARM 
- Для Windows:
   - WIN (loadparm) — для касс АТОЛ, ФРАНТОЛ, СБИС. Уточните у техотдела клиента.
   - WinCLI (sbpilot) — для 1С и др. (спросите у техотдела: sbpilot или loadparm).
   *В инструкции используется loadparm.*

------------------------------------------------------------
4. НАСТРОЙКА ПРОГРАММЫ (ФАЙЛ PINPAD)
------------------------------------------------------------
1. Узнайте COM-порт терминала:
   - Диспетчер устройств -> Порты (COM и LPT) -> Ingenico Telium (запомните номер, например COM12).
2. В папке C:\sc552 найдите файл Pinpad (конфигурация терминала).
3. Откройте его и найдите параметр ComPort.
4. Пропишите номер порта (например, ComPort=12).
5. Параметр ShowScreen: уберите скобки, поставьте 1, чтобы окно программы отображалось на ПК.
6. Сохраните и закройте файл.

------------------------------------------------------------
5. ЗАПУСК И ОБНОВЛЕНИЕ (LoadParm)
------------------------------------------------------------
1. Запустите программу LoadParm.
   - Если все верно, откроется окно на ПК и начнется обновление конфигурационных файлов (до 5 минут).
   - В окне будут видны версии библиотек — они должны совпадать с версией ПО в пинпаде.
2. Выполните удаленную загрузку параметров:
   - «Настройки» -> «Удаленная загрузка» -> Enter.
   - Введите код авторизации из заявки.
   - Сверьте TID (из заявки).
   - Терминал перезагрузится и загрузит параметры.

------------------------------------------------------------
6. ТЕХОБСЛУЖИВАНИЕ: УДАЛЕНИЕ И ЗАГРУЗКА КЛЮЧЕЙ
------------------------------------------------------------
1. Снова запустите LoadParm.
2. «Настройки» -> «Техобслуживание» -> пароль 878787 -> Enter.
3. Удаление ключей:
   - Переместитесь вниз -> «Ключи» -> «Удаление».
   - Удалите ТОЛЬКО два ключа (PIN и МАC):
     * Ключ PIN (подтвердить)
     * Ключ МАC (подтвердить)
4. «Сверка итогов»:
   - Если регформа настроена верно, ключи загрузятся автоматически.

------------------------------------------------------------
7. ТЕСТОВЫЕ ОПЕРАЦИИ (ЧЕРЕЗ КОМАНДНУЮ СТРОКУ)
------------------------------------------------------------
- В папке C:\sc552 в адресной строке напишите cmd (Enter).

  ОПЛАТА (карта):
  > loadparm 1 100        (1 - оплата, 100 - сумма в копейках)
  Приложите карту.
  > type p                (показать чек)

  ОТМЕНА (чип/карта):
  > loadparm 8 100        (8 - отмена)
  Вставьте карту в терминал (проверка чипа).
  > type p

  ОПЛАТА и ОТМЕНА (по QR-коду с экрана терминала):
  > loadparm 1 100 (оплата по QR)
  > loadparm 8 100 (отмена по QR)

  СВЕРКА ИТОГОВ:
  > loadparm 7
  > type p

  ЧЕК ПОМОЩИ (важно для заявки):
  > loadparm 36
  > type p        (содержит все версионности, сфоткать)

  ПИНГ:
  > loadparm 47 2
  > type p

  *ВСЕ ЧЕКИ сфоткать полностью, не обрезать. Чек помощи при необходимости разделить на 2 фото.*

------------------------------------------------------------
8. РЕГИСТРАЦИЯ АГЕНТА
------------------------------------------------------------
1. Из папки терминала скопируйте файлы из папки «Агент».
2. Вставьте их в C:\sc552 с заменой файлов.
3. Откройте КОМАНДНУЮ СТРОКУ ОТ ИМЕНИ АДМИНИСТРАТОРА.
4. Перейдите в папку C:\sc552:
   > cd C:\sc552      
5. Последовательно выполните команды:
   > regsvr32 sbrf.dll
   > regsvr32 sbrfcom.dll
   > agent.exe /reg
   (После каждой команды должно быть сообщение об успехе)
6. Сфоткайте результаты регистрации для заявки.
7. Запустите LoadParm и нажмите «Сверка итогов» — агент включится в работу.
8. В папке появится подпапка Update с данными для терминала.

------------------------------------------------------------
9. ЗАВЕРШЕНИЕ
------------------------------------------------------------
- Интеграция терминала с ПК завершена.
- Все чеки, скриншоты регистрации агента и сверок приложить в заявку.
- Заявка может быть закрыта без риска получения фроста по версиям или агенту.
