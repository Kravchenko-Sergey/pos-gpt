=== ДЕФЕКТОВКА PAX S90 ===

--- ВНЕШНИЙ ОСМОТР ---

1. Проверьте отсутствие посторонних предметов в ридерах
2. Осмотрите корпус на наличие повреждений и загрязнений
3. Проверьте целостность контактов SIM-слота и Ethernet-порта
4. Оцените общее состояние терминала

--- ВКЛЮЧЕНИЕ ТЕРМИНАЛА ---

1. Включите терминал и проверьте процесс загрузки
2. Убедитесь в отсутствии сообщений об ошибках: TAMPER, TAMPERED DEVICE, TAMPERING DETECTED, PED TAMPERED, PLS INPUT PWD
3. При обнаружении ошибок — терминал неисправен, требуется отправка в сервисный центр

--- МАГНИТНЫЙ РИДЕР ---

1. Зажмите кнопку func при включении
2. Выберите: Module Test → Magcard Reader
3. Проведите карту — должен отображаться READ TRACK OK или Status=0

--- ЧИПОВЫЙ РИДЕР ---

1. Зажмите кнопку func при включении
2. Выберите: Module Test → Card Readers → User Card
3. Вставьте карту — должен отображаться OK или Card test done

--- БЕСКОНТАКТНЫЙ РИДЕР ---

1. Зажмите кнопку func при включении
2. Выберите: Module Test → Card Readers
3. Выберите A Type Card или поднесите карту при появлении PLS SWIPED CARD
4. Должен отображаться read succeeded или A CARD TEST OK

--- КЛАВИАТУРА ---

1. Зажмите кнопку func при включении
2. Выберите: Module Test → KEY TEST → User Interface → Keyboard
3. Нажмите все клавиши — значения должны отображаться на экране
4. Для выхода нажмите дважды 0 или 1

--- GPRS ---

1. Зажмите кнопку func при включении
2. Выберите: Module Check → WNET
3. Нажимайте Enter до появления WNET DIALING
4. Должна появиться шкала уровня сигнала

--- ETHERNET ---

1. Визуально проверьте разъём на повреждения
2. Подключите терминал к ноутбуку патч-кордом
3. Проверьте сетевые подключения ноутбука
4. Должно появиться новое сетевое подключение
