=== КОД ОШИБКИ: 16 ===

Возможная причина (отображается на экране терминала): Не поддерживается в данном режиме устройства

Описание: Частая проблема при розничной продаже — уведомление «Не поддерживается в данном режиме устройства».

--- ДЕЙСТВИЯ ИНЖЕНЕРА ---

1. Закройте смену. Как вариант можно сделать общее гашение: «Настройки / Обслуживание кассы / Доп. операции / Общее гашение»

2. Сбросьте настройки — сделайте технологическое обнуление: «Настройки / Обслуживание кассы / Доп. операции / Технологическое обнуление». Если прошивка ККТ старше 4544, аппарат нужно будет вскрыть (не пытайтесь сделать сами, обратитесь в техподдержку)

3. Проверьте адрес и порт ОФД в разделе «Настройки ОФД»

4. Если онлайн-касса зарегистрирована на несколько ОСНО, выберите ОСНО по умолчанию: «Настройки / Обслуживание кассы / Доп. операции / Выбор СНО по умолчанию»

5. Для проверки работы кассы осуществите продажу

--- ДОПОЛНИТЕЛЬНЫЕ ДЕЙСТВИЯ (ЕСЛИ КАССА ВСЁ РАВНО НЕ РАБОТАЕТ) ---

1. Создайте резервную копию: «Ещё / Резервное копирование»

2. Вытащите аккумулятор на пару минут, затем заново установите (актуально для моделей СТ5 / СТ7.3 / СТ10)

3. Установите актуальные дату и время: «Настройки / Обслуживание кассы / Дополнительные операции / Установка даты и времени»

4. Закройте смену или сделайте Общее гашение: «Настройки / Обслуживание кассы / Дополнительные операции / Общее гашение»

5. Сделайте обнуление настроек: «Настройки / Обслуживание кассы / Дополнительные операции / Технологическое обнуление»

6. Проведите продажу для проверки работы кассы

--- ПРИМЕЧАНИЕ ---

Важно: после технического обнуления возможно, что на чеке будут пробиты неверные дата и время. Поэтому обязательно заранее настройте их.
