=== ПРОШИВКА Ingenico ICT220 / ICT250 ===

ЧТО ПОНАДОБИТСЯ:

1. Флешка, отформатированная в FAT32
2. Папка с файлами: https://disk.yandex.ru/d/-L6rGWh_bGYcog

ПРОЦЕСС ПРОШИВКИ ПОШАГОВО:

Шаг 1: Включите терминал

Шаг 2: Присоедините флешку

Шаг 3: Одновременно нажмите желтую кнопку и точку. Держите до перезагрузки

Шаг 4: Зажмите F4. На экране должна появиться надпись USB MASS STORAGE

Шаг 5: Дождитесь появления меню. Выберите FLASHCLEAN

Шаг 6: Дождитесь появления надписи Software Activate OK. Терминал перезагрузится

Шаг 7: После перезагрузки на экране должна появиться надпись LLT

Шаг 8: Одновременно нажмите желтую кнопку и точку. Держите до перезагрузки

Шаг 9: Зажмите F4. На экране должна появиться надпись USB MASS STORAGE

Шаг 10: Дождитесь появления меню. Выберите файл ОС — 9_32_3V

Шаг 11: Дождитесь загрузки файлов. Появится надпись Software Activate OK. Терминал перезагрузится

Шаг 12: Одновременно нажмите желтую кнопку и точку. Держите до перезагрузки

Шаг 13: Зажмите F4. На экране должна появиться надпись USB MASS STORAGE

Шаг 14: Дождитесь появления меню. Выберите файл Stempel

Шаг 15: Дождитесь загрузки файлов. Появится надпись Software Activate OK. Терминал перезагрузится

Шаг 16: Зажмите F4. На экране должна появиться надпись USB MASS STORAGE

Шаг 17: Дождитесь появления меню. Выберите файл с моделью терминала

Шаг 18: Появится надпись Software Activate OK. Терминал перезагрузится

Шаг 19: Дождитесь загрузки UPOS
