=== ПРОШИВКА Verifone VX520 ===

ЧТО ПОНАДОБИТСЯ:

1. Флешка, отформатированная в FAT32
2. Папка с файлами прошивки: https://disk.yandex.ru/d/Ym5ZizF5xbCEHQ

ПРОЦЕСС ПРОШИВКИ ПОШАГОВО:

Шаг 1: Включите терминал и подключите флешку

Шаг 2: Для входа в меню зажмите зелёную кнопку и цифру 7

Шаг 3: Введите пароль: 1 alpha alpha 6 6 8 3 1

Шаг 4: Выберите: Memory Function → Clear Mem

Шаг 5: Повторно введите пароль

Шаг 6: Выберите Clear all Groups и подтвердите кнопкой 2

Шаг 7: После очистки нажмите красную кнопку

Шаг 8: Выберите Restart для перезагрузки

Шаг 9: При предложении загрузить все файлы выберите NO

Шаг 10: Выберите нужную прошивку и нажмите YES

Шаг 11: Остальные шаги пропустите, нажимая NO

Шаг 12: Извлеките флешку по запросу системы

Шаг 13: Нажмите зелёную кнопку

Шаг 14: Дождитесь завершения прошивки и загрузки UPOS
