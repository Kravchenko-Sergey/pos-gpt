=== ПРОШИВКА PAX D230 ===

ЧТО ПОНАДОБИТСЯ:

1. Ноутбук
2. Папка с программой прошивки: https://disk.yandex.ru/d/jt5WS_fWPZHaUg
3. Комплектный USB-кабель
4. Драйвер устройства: https://disk.yandex.ru/d/0XtlL9aZeqnAHA

ПРОЦЕСС ПРОШИВКИ ПОШАГОВО:

Шаг 1: Проверьте наличие драйвера в Диспетчере устройств. При необходимости установите драйвер

Шаг 2: Подключите терминал к компьютеру USB-кабелем

Шаг 3: Многократно нажимайте кнопку 2 для входа в системное меню

Шаг 4: Выберите System Config и введите пароль 123456

Шаг 5: Нажмите 4 для включения XCB Service

Шаг 6: Выберите пункт 3 — USB

Шаг 7: Запустите программу Starter из папки прошивки

Шаг 8: Введите номер COM-порта (можно посмотреть в Диспетчере устройств)

Шаг 9: Выберите нужную версию прошивки

Шаг 10: Дождитесь завершения загрузки

Шаг 11: На терминале выберите XCB Service → Close Service

Шаг 12: Несколько раз нажмите красную кнопку и выключите терминал

Шаг 13: После перезагрузки дождитесь загрузки UPOS
