=== ПРОШИВКА PAX SP30 ===

ЧТО ПОНАДОБИТСЯ:

1. Комплектный провод подключения
2. Блок питания micro-USB
3. Флешка, отформатированная в FAT32
4. Переходник USB 2.0 (мама) — USB 2.0 (мама)
5. Файл прошивки: https://disk.yandex.ru/d/4fC0Y01j_Gu1-A

ПРОЦЕСС ПРОШИВКИ ПОШАГОВО:

Шаг 1: Подключите флешку через переходник к проводу пинпада

Шаг 2: Подключите питание — должен появиться статус U-DISC_DOWNLOAD

Шаг 3: Выберите файл прошивки !SB_SP30_33.33

Шаг 4: В меню выберите: DELETE APP → Delete all app

Шаг 5: Затем выберите DOWNLOAD ALL для начала загрузки

Шаг 6: Нажмите красную кнопку — появится DOWNLOAD OVER

Шаг 7: Ещё раз нажмите красную кнопку для перезагрузки

Шаг 8: Во время перезагрузки извлеките флешку

Шаг 9: Дождитесь полной загрузки UPOS
